# DevOps Assistant

A practical DevOps learning project featuring useful Linux, Docker and networking commands with clear explanations.

## Purpose

This project is designed to evolve through a complete DevOps and SecDevOps learning journey.

The application itself is intentionally simple so that the focus remains on infrastructure, automation, deployment, monitoring and security.

## Version 1

Current features:

* Static website
* Linux command reference
* Docker command reference
* Networking command reference
